Blue X-ray 프로그램이 실행되지 않는 경우

2015-12-02 16:24:04

작업관리자 → 서비스 BluXrayServ가 실행이 되었는지 확인.
작업관리자 → 프로세스 BlueX-ray.exe가 실행이 되었는지 확인.
바탕화면 → Blue X-ray 더블클릭.
위 항목들을 확인하여 주셨음에도 문제가 있으시다면, 관리자에게 문의해주시기 바랍니다.

웹페이지 화면이 깨져서 보이는 경우

2015-12-02 16:19:22

웹페이지에서 [도구] → [호환성 보기 설정] → 현재 웹사이트 추가 → 닫기 해주시면 됩니다.
